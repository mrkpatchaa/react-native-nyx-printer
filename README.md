# react-native-nyx-printer

Use Nyx devices Printer such as NB55 in React Native

## Installation

```sh
npm install react-native-nyx-printer
```

## Usage

```js
import { multiply } from 'react-native-nyx-printer';

// ...

const result = await multiply(3, 7);
```
